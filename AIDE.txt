Voici les étapes pour exécuter le programme avec la méthode dans un terminal: 
- Ouvrir le terminal
- Parcourir jusqu'au dossier src dans WorldOfSprites dans le terminal ( avec la commande cd, par exemple cd Documents/2I013/WorldOfSprites/src )
- Compiler le programme avec la commande suivante : javac *.java
- Exécuter le programme avec la commande suivante : java World
- Le programme s'exécute

Attention : 
Si le répertoire pictures ne se trouve pas dans src, il est impossible d'exécuter le programme avec java World
Il faut ainsi déplacer le dossier pictures dans src avant d'exécuter le programme

Méthode avec Eclipse :
- Ouvrir Eclipse IDE
- Parcourir dans File ( en haut à gauche ), puis Open Projects from File System…
- A la ligne Import source, selectionner Directory…
- Parcourir jusqu’au programme décompressé et sélectionner WorldOfSprites
- Dans Project Explorer, parcourir dans WorldOfSprites, src, (default package)
- Faire un clique droit sur World.java, puis Run As, Java Application
- Le programme s'exécute

Attention !!! :
Si le programme exécuté est trop petit ou trop grand, il faut modifier les valeurs se trouvant dans World.java
Voici comment faire :
- Accéder au dossier src
- Ouvrir avec un éditeur de texte World.java
- Modifier uniquement les valeurs aux lignes 16 et 20, qui représentent respectivement la taille du monde et la taille des images
- Après modification, sauvegarder les changements, recompiler et exécuter à nouveau le programme
